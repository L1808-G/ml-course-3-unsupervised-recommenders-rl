# ml-course-3-unsupervised-recommenders-rl
# Aprendizaje No Supervisado, Sistemas de Recomendación y Aprendizaje por Refuerzo

Este repositorio contiene laboratorios y experimentos desarrollados durante el **Curso 3 de la Especialización en Aprendizaje Automático**, impartida por DeepLearning.AI y Stanford Online.

El curso aborda técnicas avanzadas de aprendizaje automático enfocadas en la exploración de datos sin etiquetas, la construcción de sistemas de recomendación y la toma de decisiones mediante aprendizaje por refuerzo.

## Objetivos del curso
- Aplicar técnicas de aprendizaje no supervisado como agrupación, reducción de dimensionalidad y detección de anomalías.
- Construir sistemas de recomendación utilizando filtrado colaborativo y métodos basados en contenido.
- Implementar un modelo de aprendizaje por refuerzo profundo.
- Analizar el rendimiento de los modelos y reflexionar sobre consideraciones éticas en el uso de datos y algoritmos.

## Contenidos principales
- Aprendizaje no supervisado
- Clustering
- Reducción de dimensionalidad
- Detección de anomalías
- Sistemas de recomendación
- Filtrado colaborativo
- Recomendación basada en contenido
- Aprendizaje por refuerzo
- Evaluación de modelos
- Ética de datos

## Tecnologías utilizadas
- Python
- NumPy
- Scikit-learn
- TensorFlow
- Jupyter Notebook

## Enfoque de aprendizaje
Los notebooks de este repositorio priorizan la comprensión conceptual y la aplicación práctica de los algoritmos, destacando cómo estas técnicas pueden utilizarse para resolver problemas reales como exploración de datos, personalización y toma de decisiones automatizada.
